# How to use

Vim scripts are mainained by [Vundle](https://github.com/gmarik/Vundle.vim).

## Mac or Linux

```bash
git clone https://github.com/chrisyip/vim-config.git ~/.vim
cd ~/.vim
./install.sh
```

Open Vim then:

```vim
:PluginInstall
```

Command line:

```shell
vim +PluginInstall +qall
```
